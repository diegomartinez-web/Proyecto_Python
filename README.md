# Proyecto: Generador de Contraseñas Seguras con Python 
Un programa en Python que crea contraseñas aleatorias personalizables (longitud, mayúsculas, números, símbolos) y evalúa su fortaleza. Ideal para aprender manejo de strings, módulo random y buenas prácticas de seguridad básica. 

# Tecnologías utilizadas:
Python 3 (lenguaje principal)

- Módulos estándar de Python:
- random (para generación aleatoria)
- string (para manejo de caracteres)
